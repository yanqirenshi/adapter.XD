  |     |
  |     +--- graphics/
  |     |      |
  |     |      `--- graphicContent.agc (json)
  |     |

  +-- sharing/

  |     `--- sharing.json

  +-- manifest (json)
  |
  +-- mimetype (text)

  +-- preview.png
